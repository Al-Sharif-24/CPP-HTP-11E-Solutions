# Exercsie 3.11: What does this program do?

**What does the following program print?**

```cpp
#include <iostream>
using namespace std;

int main() {
    int count{1};

    while (count <= 10) {
        cout << (count % 2 == 1 ? "****" : "++++++++") << "\n";
        ++count;
    }
}
```
