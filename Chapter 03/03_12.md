# Exercise 3.12: What Does This Program Do?

**What does the following program print?**

```cpp
#include <iostream>
using namespace std;

int main() {
    int row{10};

    while (row >= 1) {
        int column{1};

        while (column <= 10) {
            cout << (row % 2 == 1 ? "<" : ">");
            ++column;
        }

        --row;
        cout << "\n";
    }
}
```
