# Exercise 3.14: Another Dangling else problem

```cpp
    if (x < 10)
    if (y > 10)
    cout << "*****\n";
    else
    cout << "#####\n";
    cout << "$$$$$\n";
```
when `x = 9` and `y = 11`
```txt
*****
$$$$$
```

when `x = 11` and `y = 9`
```txt
$$$$$
```
