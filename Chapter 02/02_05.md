# Exercise 2.5

**Identify and correct the errors in each of the following statements (assume that the
statement `using std::cout;` is used):**

- a)
```cpp
    if (c < 7); {
        cout << "c is less than 7\n";
    }
```
```cpp
    if(c < 7) {
        cout << "c is less than 7\n";
    }
```

- b)
```cpp
    if(c => 7) {
        cout << "c is equal to or greater than 7\n";
    }
```
```cpp
    if(c >= 7) {
        cout << "c is equal to or greater than 7\n";
    }
```
