# Exercise 4.6: What does the Code do?

**What does the following program do?**

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i{1}; i <= 10; i++) {
        for (int j{1}; j <= 5; j++) {
            cout << '@';
        }

        cout << '\n';
    }
}
```

This program prints a rectangle of `@` with 10 rows and 5 columns
