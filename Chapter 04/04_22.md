# Exercise 4.22: What does this Code do?

**What does the following program segment do?**

```cpp
for (int i{1}; i <= 5; i++) {
    for (int j{1}; j <= 3; j++) {
        for (int k{1}; k <= 4; k++) {
            cout << '*';
        }
        cout << '\n';
    }
    cout << '*';
} 
```

The output is 
```txt

*****
```
