# Solutions to Exercise 5.14 (Celsius and Fahrenheit Temperatures)

```cpp
int celsius(int fahrenheit) {
    return (fahrenheit - 32) * 5 / 9;
}

int fahrenheit(int celsius) {
    return celsius * 9 / 5 + 32;
}
```
