# Exercise 5.25: Fibonacci Series: Iterative Solution

**Write a nonrecursive version of the function `fibonacci`.**

```cpp
    void fibonacci(int n) {
        int x = 1;
        int y = 1;

        std::cout << x << " " << y << " ";

        for(int i = 1; i <= n - 2; ++i) {
            std::swap(x, y);

            x = x + y;

            std::cout << x << " ";
        }
    }
```
