# Exercise 5.1: Value of a Variable

**Show the value of x after each of the following statements is performed:**

- a)
```cpp
    x = fabs(7.5);
```

x = 7

- b)
```cpp
    x = floor(7.5);
```

x = 7.5

- c)
```cpp
    x = fabs(0.0);
```

x = 0

- d)
```cpp
    x = ceil(0.0);
```

x = 1

- e)
```cpp
    x = fabs(-6.4);
```

x = 6.4

- f)
```cpp
    x = ceil(-6.4);
```

x = -6

- g)
```cpp
    x =  ceil(-fabs(-8 + floor(-5.5)));
```

x = ceil(-fabs(-8 + -6))
x = ceil(-fabs(-14))
x = ceil(-14)
x = -14
