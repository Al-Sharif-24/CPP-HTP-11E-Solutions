# Exercise 5.18: Reverse Digits

**Write a function that takes an integer value and returns the number with its digits reversed.**

```cpp
    int reverse(int n)
    {
        int reversed = 0;

        while (n != 0) {
            int digit = n % 10;
            reversed = reversed * 10 + digit;
            n /= 10;
        }

        return reversed;
    }
```
